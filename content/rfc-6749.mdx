# OAuth 2.0이란?

RFC 6749는 OAuth 2.0 프로토콜에 대한 표준입니다.

## 핵심 개념
1. **Resource Owner**: 리소스 소유자 (유저)
2. **Client**: 애플리케이션
3. **Authorization Server**: 권한 서버

> "Access Token은 열쇠지 신분증이 아니다."

## 주요 흐름
Authorization Code Grant 방식이 가장 많이 쓰입니다.
